# Mini Course Management System

> **Team:** Kavy Khilrani · Ruchi Pawar · Aarjica Talati  
> **Subject:** Database Management Systems (DBMS) College Project  
> **Last Updated:** March 2026

A database-driven Java Swing application for managing academic activities — course registration, assignment submission, grading, and student–instructor interaction.

---

## 📋 Features

### 🎓 Student View
| Feature | Description |
|---|---|
| **My Courses** | View all enrolled courses with instructor and department info |
| **Assignments** | See all assignments across courses with deadlines and submission status (✅ / ❌) |
| **Submit Assignment** | Dropdown of pending assignments — select and click Submit |
| **My Grades** | View graded submissions with marks and instructor feedback |

### 👨‍🏫 Instructor View
| Feature | Description |
|---|---|
| **Manage Courses** | View all your courses with enrolled student count; add new courses (ID auto-generated); **delete a selected course** (cascades to assignments, submissions, grades and enrollments) |
| **Manage Assignments** | View all assignments; create new ones; **delete a selected assignment** (cascades to submissions and grades) |
| **Grade Submissions** | View all student submissions; click a row to auto-fill the grading form; save or update grades with feedback; **remove a grade** (submission is kept, only the grade record is deleted) |

### 🔀 Profile Switching
Switching between Student and Instructor views automatically updates the profile name in the top navigation bar:
- **Student View** → `Kavy Khilrani | Student`
- **Instructor View** → `Dr. Jashma Suresh | Instructor`

## 🚀 How to Run

### Prerequisites
- **Java JDK 8 or above** must be installed.
- Verify with: `java -version` in Command Prompt.
- An active internet connection is required **only on the first run** to download the JDBC driver.

### ▶️ Option 1 — Recommended (run.bat)
Double-click **`run.bat`** in the project folder.

It will automatically:
1. Download `sqlite-jdbc.jar` into `lib/` (first run only).
2. Create the `bin/` folder and compile all `.java` source files.
3. Launch the application window.

### ▶️ Option 2 — From VS Code
The `.vscode/settings.json` file is included so VS Code automatically adds `lib/*.jar` to the classpath.
> ⚠️ Make sure the jars exist in `lib/` first — run `run.bat` once to download them, then you can launch from VS Code.
